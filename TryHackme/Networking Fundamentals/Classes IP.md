***Historia*** : As classes surgiram para diferenciar os enderecos de IP's. A causa foi a quantidade de computadores que surgiram por rede.

***Diferenciar classes***: Para diferenciar a classes uma da outras voce precisa saber os intervalos numericos que representam cada classe.

```ad-important
title: Tipos de classes IP
collapse: open

**Classe A**  - Se o primeiro octeto estiver entre 0 e 127.

**Classe B** - Se o primeiro octecto estiver entre 128 e 191.

**Classe C** - Se o primeiro octecto estiver entre 192 e 223.

**Classe D** - Se o primeiro octecto estiver entre 224 e 239.

**Classe E** - Se o primeiro octecto estiver entre 240 a 255.

Lembrete: As classes A,B,C sao usadas pela internet enquanto as classes D,E sao para fins especificos.

```
